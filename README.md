## 🎨 Design Features

### Designinspiration
Websitet er designet med inspiration fra danske mainstream sushi restauranter som:
- **Sticks'n'Sushi** - Moderne skandinavisk æstetik
- **Sushi Anaba** - Minimalistisk design
- **Bar'sushi** - Kreativ præsentation

### Designelementer
- **Farvepalette:** Sort/hvid base med røde accenter (#ff4444)
- **Typography:** Moderne sans-serif skrifttyper
- **Layout:** Responsivt grid-system
- **Animationer:** Smooth hover-effekter og transitions
- **Komponenter:** Cards, gradients, og moderne UI-elementer

## 📱 Funktioner

### Hovedside (index.html)
- Restaurant selektor med kort for hver lokation
- Moderne hero sektion
- "Om os" sektion

- Responsiv footer

## 🚀 Sådan bruges websitet

1. **Åbn index.html** i en browser
2. **Vælg restaurant** ved at klikke på et af kortene
3. **Naviger** mellem sektioner ved hjælp af top-menu
4. **Book bord** eller **bestil takeaway** via formularer
5. **Kontakt** restauranten via kontakt sektionen
